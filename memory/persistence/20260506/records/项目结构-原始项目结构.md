# 原始项目结构（归档）

## 日期

2026-05-06（迁移整理）

## 背景

`docs/plan/原始项目结构.md` 记录了当时的项目目录树，用于对齐“当下实际结构”。

## 结论 / 事实

- 该结构是一份“当时快照”，并不一定与当前仓库实际一致（后续可能经历了重构/迁移）。
- 目录树对快速定位模块很有价值，应作为持久记忆保留，并在需要时对照当前 `src/` 实际结构更新。

## 影响 / 下一步

- 任何大规模重构后，建议补一份新的结构快照到当日 `memory/persistence/<date>/records/`，并在索引中串起来。
